*Description:
This program uses AI to detect ASL hand signs and displays them on a camera as well as in a text box.

The contents of the text box and be saved, opened from a file and deleted entirely. Additionally, users are capable of deleting the most recent letter added to the text box in the event of misinterpretations.

the AI model and its training sets are imported from ultralytics and the GUI is built entirely using tkinter.
